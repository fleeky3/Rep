## top 명령어
top 명령어는 시스템의 실시간 상태를 모니터링하는 명령어임
CPU, 메모리, 프로세스 등 시스템 자원 사용량을 지속적으로 갱신하며 보여줌

### 특징
- CPU 및 메모리 사용량 실시간 확인함
- 프로세스 우선순위, 부하 상태 확인함
- k 명령으로 top 화면에서 프로세스 종료 가능함

### 사용 예시
top
top -u username # 특정 사용자 프로세스만 표시
top -p 1234 # 특정 PID만 추적
top -n 5 # 5번 업데이트 후 종료

### 주요 옵션
- -u USER : 특정 사용자 프로세스만 표시
- -p PID : 특정 PID만 추적
- -n N : N번 업데이트 후 종료
- k : top 화면에서 특정 PID 종료 (실행 중)
- q : top 종료
